## 環境構築

クローンして、ローカルのnode_moduleをインポートしましょう。

```
$ git clone <url> && cd <repository-name>
$ yarn install
```

Dockerのビルドし、Docker内のnode_moduleをインポートしましょう。

```
$ docker-compose build

$ docker-compose run --rm app yarn install
```

アプリの立ち上げ

```
$ docker-compose build
```
